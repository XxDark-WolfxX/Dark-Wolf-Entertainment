Project: Browser Malware Detector

A Chrome Extension built to proactively block malicious network requests and scan content for signatures of known threats like ShadyPanda and GlassWorm.

Key Features:

  • Dynamic IOC Fetching: Pulls real-time threat data from AbuseIPDB and URLhaus.
  • `webRequest` Blocking: Uses a background service worker to block connections based on IP/Domain IOCs.
  • Content Scanning: Injecting client-side scripts to check page source for obfuscated payload signatures (e.g., `document.write(atob(`).
